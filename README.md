# 🌸 Iris Species Classification

## 📌 Project Overview
This project classifies iris flowers into species based on:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 📊 Dataset

Classic Iris dataset with 3 classes:
- Setosa
- Versicolor
- Virginica

Target:
- Species
